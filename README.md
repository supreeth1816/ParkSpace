# ParkSpace

ParkSpace - Smart parking monitoring and management app

- Firebase Authentication
- Google Maps API
- Dynamic Pricing
- Parking status tracking using IOT

- NodeMCU, Ultrasonic sensor
- Arduino code for parking status monitoring
